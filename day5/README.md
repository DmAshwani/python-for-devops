# Day 5 — Configuration Management & Data Formats

- `config_loader.py` — load JSON/INI (YAML optional) with environment substitution
- `exercises_day5.py` — merge configs and validate required fields

Example:
`python config_loader.py --file config.json --show`

Tip: install PyYAML (`pip install pyyaml`) if you want to load YAML configs.
